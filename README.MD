### Paytm



1. Clone the project
```
    https://github.com/abhishekreddywadi/paytm.git
```
2. Move into the directory
```
    cd paytm/backend
```
3.  install dependencies
```
   npm install
```
4.  Move into the directory
```
    cd paytm/frontend
```
5.  install dependencies
```
    npm install
```
6.  run the server
```
    npm run dev
```
